# Development Branch to Create Scripts (using OpenCV 4.6.0) before transfering to IMX project
Testing pull Clion

## Install Dependencies
- If you have already cloned the repository, you can execute `bash path/to/this-repository/scripts/install_dependencies.sh`
- If you have not already cloned this repository, you can copy and paste into the terminal the commands in [this file](/scripts/install_repository.sh)

## Building
The command to run when your current directory is the root folder of this repository, `mkdir build; cmake -Bbuild -DCMAKE_BUILD_TYPE=Release`
